# Slot Machine Domain Research Notes

## Computer System and Core Mechanics
* Reels: These are the vertical columns that spin when the game starts. Most digital games use either three or five reels.
* Symbols: These are the images on the reels. A win happens when specific symbols line up in a certain way.
* Random Number Generator: This is a computer program that picks a new number every split second. It never stops running even when nobody is playing the game. When a player clicks the spin button, the system grabs the most recent number to decide if the spin wins or loses.
* Virtual Reels: Modern games do not use simple physical wheels. Instead, they use a list of many numbers in the computer memory. The computer maps these numbers to the symbols you see on the screen. This allows the game to have millions of different outcomes so it can offer very large jackpots.
* Independence: Every single spin is a separate event. The machine does not remember if a player just won or lost. A big win on one spin does not change the chances of winning on the next spin.
* Stopping Logic: In digital slots, the system chooses the result first and then moves the reels to match that result.

## Money and Payout Rules
* Paylines: These are the specific paths across the screen that the game checks for matching symbols. Some games only check straight lines, while others check zigzags or dozens of different patterns.
* Payout Table: This is a chart that shows how much each combination of symbols is worth. 
* Bet Levels: Players can choose how much money to bet per line. The total bet is the cost per line multiplied by the number of active paylines.
* Return to Player: This is the percentage of all bets that the game is programmed to pay back over a very long time. For example, if a game has a 96 percent return, it will eventually pay back 96 dollars for every 100 dollars bet.
* Volatility: This measures how much risk is in the game. A high volatility game has rare wins but the prizes are usually very large. A low volatility game pays out small prizes very often, which helps a player play for a longer time.
* House Edge: This is the opposite of the return percentage. It is the small mathematical advantage that ensures the casino or game owner makes money over time.

## Special Symbols and Gameplay Features
* Wild Symbols: These symbols act like a joker or a blank card. They can turn into almost any other symbol to complete a winning line. Sticky wilds are special because they stay on the screen for several spins in a row.
* Scatter Symbols: These are unique because they do not have to be on a payline to work. If enough of them appear anywhere on the screen at the same time, they usually trigger a bonus round or a free spin feature.
* Multipliers: These are special numbers that increase the value of a win. If a win is worth 10 credits and a two times multiplier is active, the player gets 20 credits instead.
* Free Spins: This feature lets the player spin the reels without spending any of their own credits. Any money won during these spins is usually added to the player balance at the end.
* Cascading Wins: This is a modern feature where winning symbols disappear from the screen after a win. New symbols fall down from the top to fill the empty spaces, which can create a second win on the same spin.
* Near Miss: This is when the symbols almost form a winning line so it seems like a win is gonna happen soon

## Common Features in Slot Apps
* Auto Spin: This allows the player to set the game to spin itself for a certain number of turns.
* Balance Tracker: This displays the current amount of money or credits the player has left.
